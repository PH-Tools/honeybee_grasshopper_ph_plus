# honeybee_grasshopper_ph_plus/ — the GH plugin package (deployed artifact)

The canvas-facing half: thin component wrappers, compiled user objects, and icons. Logic lives in `../honeybee_ph_plus_rhino/`. See `../context/ARCHITECTURE.md`.

## Contents

- `src/` — the `HBPH+ - <Name>.py` component **wrappers** (one per component). Thin GHPython shims: GPL header + Args/Returns docstring, set `ghenv.Component.Name`, build `IGH`, instantiate the matching `GHCompo_*` from `../honeybee_ph_plus_rhino/gh_compo_io/`, call `.run()`. No logic here.
- `user_objects/` — the matching compiled `.ghuser` user objects (build output; regenerated on release — do not hand-edit).
- `icons/` — component icons: `AI/` (Illustrator source) + `PNG/` (exports).

## Notes
- Wrapper filename, `ghenv.Component.Name`, and the `_component_info_.py` key must all match `"HBPH+ - <Name>"`.
- IronPython-2.7-safe only. See `../context/CODING_STANDARDS.md`.
